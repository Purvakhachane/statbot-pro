# StatBot Pro

## AI-Powered Data Analysis Assistant

StatBot Pro is an intelligent data analysis platform that allows users to upload datasets, explore data visually, and interact with their data using natural language queries.

The goal of this project is to simplify data analysis for non-technical users by combining data visualization, query parsing, and AI-assisted insights into a single platform.

---

## 🚀 Features

### 📁 Dataset Upload & Management

* CSV file upload
* Dataset preview
* File validation
* Dataset statistics

### 📊 Visualization Module

* Interactive charts
* Bar Charts
* Line Charts
* Pie Charts
* Correlation Analysis
* Missing Value Analysis

### 🤖 AI Query Engine

* Natural language query support
* Query mapping
* Query parsing
* Pandas operation generation
* Aggregation support
* Filtering support
* GroupBy analysis

### 📈 Analytical Insights

* Dataset summary
* Missing value detection
* Data quality analysis
* Dataset health scoring
* Statistical analysis
